# Prueba de Git - 10 Preguntas de Opción Múltiple

Esta prueba evalúa tus conocimientos básicos de Git después de ver el video del curso. Responde cada pregunta seleccionando la opción correcta (A, B, C o D). Al final, revisa las respuestas correctas.

## Pregunta 1: ¿Qué es Git?
A) Un sistema operativo  
B) Un sistema de control de versiones distribuido  
C) Un editor de texto  
D) Una base de datos  

**Respuesta correcta: B**

## Pregunta 2: ¿Cuál es el comando para instalar Git en Ubuntu?
A) sudo apt install python  
B) sudo apt update && sudo apt install git  
C) brew install git  
D) npm install git  

**Respuesta correcta: B**

## Pregunta 3: ¿Qué representa el área de staging en Git?
A) El lugar donde se almacenan los commits finales  
B) Una zona de preparación para decidir qué cambios incluir en el próximo commit  
C) El directorio de trabajo local  
D) El repositorio remoto  

**Respuesta correcta: B**

## Pregunta 4: ¿Qué comando se usa para ver el estado actual del repositorio?
A) git log  
B) git status  
C) git diff  
D) git add  

**Respuesta correcta: B**

## Pregunta 5: ¿Cuál es la función de `git add archivo.txt`?
A) Crear un nuevo commit  
B) Agregar el archivo al área de staging  
C) Ver el historial de commits  
D) Eliminar un archivo del repositorio  

**Respuesta correcta: B**

## Pregunta 6: ¿Qué diferencia hay entre el directorio de trabajo y el repositorio?
A) El directorio de trabajo es remoto, el repositorio es local  
B) El directorio de trabajo contiene archivos modificados, el repositorio almacena el historial confirmado  
C) Ambos son lo mismo  
D) El repositorio es temporal, el directorio de trabajo es permanente  

**Respuesta correcta: B**

## Pregunta 7: ¿Cómo se hace un commit en Git?
A) git add . && git commit -m "mensaje"  
B) git status  
C) git log  
D) git diff  

**Respuesta correcta: A**

## Pregunta 8: ¿Qué es una rama (branch) en Git?
A) Una copia del repositorio para trabajar en paralelo  
B) Un archivo de configuración  
C) El historial de commits  
D) Una herramienta para instalar Git  

**Respuesta correcta: A**

## Pregunta 9: ¿Qué muestra el comando `git log`?
A) Los archivos modificados  
B) El historial de commits  
C) El estado del staging  
D) Las diferencias entre archivos  

**Respuesta correcta: B**

## Pregunta 10: ¿Cuál es el propósito de `git diff`?
A) Ver las diferencias entre el working directory y el staging area  
B) Crear un commit  
C) Agregar archivos al staging  
D) Ver el historial de commits  

**Respuesta correcta: A**
